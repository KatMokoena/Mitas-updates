## Essential Concepts (5-Minute Overview)

### What is an Order?
A customer order with a deadline. Contains tasks that must be completed.

### What is a Task/Step?
Individual work items. Can depend on other tasks and have assigned resources.

### What is Critical Path?
Tasks with zero buffer time. Any delay delays the entire order.

### What are Risk Levels?
- **Green (On Track)**: Will finish before deadline
- **Amber (At Risk)**: Close to deadline (< 7 days buffer)
- **Red (Late)**: Will miss deadline

### How Does Auto-Calculation Work?
System automatically calculates when tasks should start/end based on:
- Task dependencies
- Purchase delivery dates
- Resource availability
- Order deadline

## Common Tasks - Step by Step

### Create a New Order
1. Click "Orders" in sidebar
2. Click "+ New Order"
3. Fill: Order Number, Customer, Deadline, Priority
4. Click "Create Order"

### Add a Workflow Step
1. Open an order
2. Go to "Workflow" tab
3. Click "Add Step"
4. Enter: Name, Description, Duration
5. Set dependencies (if any)
6. Assign resources (if needed)
7. Click "Create Step"

### Update Task Status
1. Open order → "Workflow" tab
2. Find the task
3. Use status dropdown
4. Select: Not Started / In Progress / Completed / Blocked

### Add a Purchase
1. Open order → "Purchases" tab
2. Click "Add Purchase"
3. Enter: Supplier, Item, Delivery Dates
4. Link to task (optional)
5. Click "Create Purchase"

### View Timeline
1. Open order → "Timeline" tab
2. Review Gantt chart
3. Check critical path (red tasks)
4. Verify projected completion vs. deadline

## Troubleshooting Quick Fixes

### Timeline seems wrong
- Click "Recalculate Timeline" button
- Check task dependencies are correct
- Verify purchase delivery dates

### Order shows as late but shouldn't
- Check all task durations are realistic
- Verify dependencies are correct
- Review purchase delivery dates

### Can't see an order
- Check filters on Dashboard
- Verify department access (if applicable)
- Check order status (may be filtered out)

### Task won't update
- Refresh the page
- Check you have edit permissions
- Verify task isn't locked

## Key Metrics to Monitor

### Dashboard Metrics
- **Total Projects**: All active orders
- **Tasks Completed Today**: Daily progress
- **Project Health Efficiency**: Overall performance %
- **On Track / At Risk / Late**: Risk distribution

### Order-Level Metrics
- **Days Until Deadline**: Countdown timer
- **Projected Completion**: Calculated finish date
- **Critical Path Tasks**: Number of critical tasks
- **Completed Steps**: Progress percentage
